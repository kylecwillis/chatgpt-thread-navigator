## Detailed description

A floating outline for long ChatGPT conversations.

Long ChatGPT conversations are powerful, but hard to navigate. ChatGPT Thread Navigator adds a small floating outline to the right side of chatgpt.com so you can jump through long threads like a document.

It automatically lists every user question in the current conversation, plus h1, h2, and h3 headings from assistant responses. Click any item to smooth-scroll directly to that part of the thread. Star the items you want to find again.

Built for researchers, developers, writers, students, and ChatGPT power users who work in long conversations and need a faster way to find earlier prompts and answers.

Features
- Floating conversation outline
- Lists user questions and prompts
- Detects assistant headings
- Smooth-scroll navigation
- Star and bookmark items per conversation
- Auto-updates while responses stream
- Light and dark mode support
- No popup
- No settings
- No dependencies
- Lightweight and fast

Privacy
- Runs locally in your browser
- Only runs on chatgpt.com
- Does not collect your conversations
- Does not send your data to a server
- No account required

## Single purpose
Display a navigable outline of the current ChatGPT conversation so users can jump to earlier prompts and assistant headings.

## Permission justifications
No permissions are requested. The extension only runs a content script on https://chatgpt.com/* and uses DOM APIs and localStorage.

## Privacy practices
- Does not collect any user data
- Does not transmit any data off the device
- No remote code is executed
